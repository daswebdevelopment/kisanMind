# Issue #1: Setup project structure and base skeleton for KisanMind

## Objective
Create a clean starter monorepo layout with:
- Vue.js 3 frontend (Vite)
- FastAPI backend with a basic health route

## Delivered scope
1. Added `frontend/` Vue 3 + Vite skeleton, router, and two starter views.
2. Added `backend/` FastAPI skeleton with `GET /health` route.
3. Added placeholders for models, services, and database module.
4. Added `.env.example` and `.gitignore` baseline for local development.
5. Updated documentation for running frontend/backend locally.

## Out of scope
- AI workflows and model integrations
- Production database wiring
- Market data integrations
